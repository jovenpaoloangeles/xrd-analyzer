# Technology Stack

## Core Frontend
- **React (TypeScript):** Primary framework for building the user interface with strong typing for scientific data structures.
- **Vite:** Modern build tool and development server for fast iteration.

## UI & Styling
- **Tailwind CSS:** Utility-first CSS framework for responsive and consistent layout.
- **shadcn/ui & MUI:** Component libraries used for modern, accessible, and professional UI elements.
- **Lucide React:** Icon set for clear visual communication.

## Data Visualization & Processing
- **Plotly.js (react-plotly.js):** High-performance charting library used for interactive XRD plots.
- **Scientific Libraries (ml.js):**
    - `ml-savitzky-golay`: For data smoothing.
    - `ml-matrix` & `ml-levenberg-marquardt`: For mathematical operations and curve fitting.
- **React Dropzone:** For efficient drag-and-drop file handling.

## Deployment & Infrastructure
- **GitHub Pages:** Platform for hosting the static web application.
- **GitHub Actions:** Automated deployment pipeline.
